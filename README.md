# 82-0

随机球队 × 随机年代 × 五次机会。组建五人阵容，冲击 82 胜完美赛季。

## 技术

Next.js 15 / TypeScript / PostgreSQL (Supabase) / Drizzle ORM / CSS

## 模式

| 模式         | 玩法                                                     |
| ------------ | -------------------------------------------------------- |
| **经典随机** | 老虎机抽球队+年代，每轮 Top20 选 1 人，5 轮后模拟 82 场  |
| **自选娱乐** | 自由选球队+年代，全量球员按评分降序，手动组建结算        |
| **圆梦模式** | 选一位无冠球星当队长，带队 82 场常规赛 → 4 轮 BO7 季后赛 |
| **双人对战** | 开房邀好友，各自组建阵容，BO7 决胜负                     |
| **挑战模式** | 工资帽 / 年代穿越 / 同队传奇 / 国际阵容                  |

## 算法

```
playerRating = 位置权重 × 年代基准 → 0-100
teamOvr      = geoMean(5人评分) × 1.1
wins         = round(82 × (teamOvr / 110) ^ 2.2)
```

单场胜负：`1 / (1 + 10^(-Δrating / 30))`，上限 95%，比分 pace ± jitter。
